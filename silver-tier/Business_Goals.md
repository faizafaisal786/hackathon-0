## Sales Pipeline Stages

| Stage | Description | AI Action |
|---|---|---|
| Lead | New inquiry received | Create task, draft intro response |
| Qualified | Budget + need confirmed | Create Plan.md with proposal outline |
| Proposal | Proposal sent | Track follow-up, draft reminder if no reply in 5 days |
| Negotiation | Back and forth on terms | Flag for owner attention, log all changes |
| Closed Won | Contract signed | Send onboarding email draft, update revenue KPI |
| Closed Lost | Deal didn't close | Log reason, send thank-you note draft |
